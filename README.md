## About Me

데이터 기반으로 문제를 정의하고, 대안을 비교하여 최적의 해결책을 도출하는 백엔드 개발자 박헌우입니다.

- Redis 자료구조 전환으로 **10만 유저 대응 메모리 40% 절감**
- Elastic Beanstalk 없이 **VPC, ALB, Auto Scaling Group을 직접 설계/구축**
